// 因子构建链路Factor pipeline

  • WAP 加权平均价格，逐档 L1→L5 五档构建weighted avg price, built across 5 levels L1→L5
  • DIj 深度不平衡，取值 [-1,+1]，作领先指标depth imbalance, range [-1,+1], a leading indicator
  • wi 买卖权重，二阶因子bid-ask weight, 2nd-order factor
  • Press 买卖压力差 = log(BidPress) − log(AskPress)pressure = log(BidPress) − log(AskPress)
  • pressDItanh 最终因子，经 tanh 压缩至 [-1,+1]final factor, tanh-compressed to [-1,+1]

// 原始模型的四点修正Four model fixes

  • 分母为零风险 → 线性 / 绝对值权重 + 极小常数 εzero-denominator risk → linear / abs weights + ε
  • 数值敏感性 → 非线性指数衰减 exp(−λ|Price−WAP|+ε)numeric sensitivity → exp(−λ|Price−WAP|+ε) decay
  • 买卖非对称性 → max / min 权重分别处理bid-ask asymmetry → separate max / min weights
  • 输出区间 → tanh 压缩，提升可解释性output range → tanh compression for interpretability

// 交易规则（三态）Trading rule (3 states)

  • pressDItanh > +0.5 → 多头 / Long (+1)pressDItanh > +0.5 → Long (+1)
  • −0.5 ~ +0.5 → 空仓 / Flat (0)−0.5 ~ +0.5 → Flat (0)
  • pressDItanh < −0.5 → 空头 / Short (−1)pressDItanh < −0.5 → Short (−1)

把量化全链路交给一群专精的 AI 智能体Handing the full quant pipeline to a team of specialist AI agents

在本地用 OpenClaw（🦞）自建一套统一 LLM 网关 + 工具集成层：把 DeepSeek、腾讯混元、Kimi 等多家模型统一挂在 models.providers 下，再通过自定义 exec 工具让模型以受限权限调用我的 DolphinDB / Python 因子与回测管线。围绕「未来量化交易智慧馆」的设定，我把它拆成五个专精角色分工协作 —— 调度中枢、数据、模型/信号、风控、情报，由 Leader_AlphaWang 调度中枢串起整条流水线。A self-hosted unified LLM gateway + tool layer built on OpenClaw (🦞): multiple providers (DeepSeek, Tencent Hunyuan, Kimi…) mounted under models.providers, with custom exec tools that let models call my DolphinDB / Python factor & backtest pipeline under limited permissions. Framed as a "future quant trading guild", it splits into five specialist roles — orchestrator, data, model/signal, risk, intelligence — strung together by the Leader_AlphaWang hub end to end.

// 统一网关Unified gateway

Gateway 默认监听 18789 端口并自带 Web Control UI；models.mode = merge 合并多家模型，按任务用 openclaw models set 在 DeepSeek / 混元 / Kimi 间切换。The gateway listens on 18789 with a built-in Control UI; models.mode = merge combines providers, switching between DeepSeek / Hunyuan / Kimi per task via openclaw models set.

// 自定义 exec 工具Custom exec tools

在 tools.exec 下封装专用脚本，把 DolphinDB / Python 的因子计算与回测包成固定入口参数，给 LLM 有限权限访问数据管线；tools.exec.security = full 启用严格沙箱。Under tools.exec I wrap DolphinDB / Python factor & backtest scripts behind fixed entry params so the LLM gets scoped pipeline access; tools.exec.security = full enables a strict sandbox.

// 反向代理 & 配对Reverse proxy & pairing

用 Nginx 终止 HTTPS 并转发到 127.0.0.1:18789，allowedOrigins 精确到内网 IP；浏览器访问需 token 配对授权，devices approve 批准设备后方可接入。Nginx terminates HTTPS and forwards to 127.0.0.1:18789; allowedOrigins is pinned to the LAN; browser access requires token pairing, approved via devices approve.
